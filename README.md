# ReportMaster

**ReportMaster** — это простое и удобное десктопное приложение для ведения личных записей. Оно поможет вам организовать финансы, задачи и события в одном месте благодаря интуитивно понятному интерфейсу с тремя основными разделами.

## Функционал

### Система авторизации
*   **Регистрация новых пользователей:** Создание учетной записи с указанием имени пользователя и пароля.
*   **Вход в систему:** Авторизация зарегистрированных пользователей для доступа к личным данным.
*   **Валидация данных:**
    *   Проверка на заполнение всех обязательных полей.
    *   Проверка уникальности имени пользователя при регистрации.
    *   Проверка корректности пароля при входе.
*   **Персональное хранение данных:** Каждый пользователь имеет собственный файл с данными, что обеспечивает приватность и разделение записей между разными людьми.

### Вкладка "Финансы"
*   **Управление доходами и расходами:** Добавление, редактирование и удаление записей о финансах.
*   **Детали записи:** Для каждой операции указывается название, категория ("Доход" или "Расход"), дата и сумма.
*   **Автоматический подсчет итога:** В нижней панели автоматически отображается общий баланс по отфильтрованному списку. Цвет меняется в зависимости от значения: зеленый для положительного, красный для отрицательного.
*   **Фильтрация:** Быстрый просмотр только доходов или только расходов с помощью выпадающего списка.

### Вкладка "Задачи"
*   **Управление задачами:** Добавление, редактирование и удаление задач.
*   **Детали задачи:** У каждой задачи есть название, категория (например, "Работа", "Дом"), дата создания, статус ("В работе", "Выполнено", "Отложено") и срок сдачи.
*   **Контроль сроков:** При добавлении или редактировании задачи предусмотрена проверка, что срок сдачи не может быть раньше даты создания.
*   **Статистика:** В нижней панели отображается общее количество задач, а также количество выполненных и находящихся в работе.

### Вкладка "События"
*   **Управление событиями:** Добавление, редактирование и удаление событий (встреч, праздников и т.д.).
*   **Детали события:** Для события указывается название, категория (предустановленные: "Праздник", "Работа", "Встреча", "Другое"), дата, место и отметка о необходимости напоминания.
*   **Умная фильтрация:** Фильтрация событий происходит по их категориям. Список категорий для фильтра автоматически формируется из уже добавленных событий.
*   **Статистика:** Отображается общее количество событий и количество событий, отмеченных для напоминания.

### Общие возможности для всех вкладок
*   **Экспорт в CSV:** Данные из любой вкладки можно экспортировать в файл CSV. Формат файла автоматически включает дату и время экспорта (например, `ReportMaster_Финансы_20260225_153045.csv`).
*   **Удаление записей:** Удалить запись можно, выделив её в таблице и нажав клавишу `Delete` или воспользовавшись контекстным меню.
*   **Редактирование записей:** Двойной клик по любой записи в таблице открывает форму для её редактирования.
*   **Автосохранение:** Все изменения автоматически сохраняются в JSON-файл при добавлении, редактировании, удалении записи или закрытии приложения.
*   **Валидация ввода:** При создании и редактировании записей производится проверка корректности вводимых данных (обязательные поля, форматы дат, числовые значения).

## Технологии

*   **Язык программирования:** C#
*   **Платформа:** .NET Framework 4.7.2
*   **Тип приложения:** Windows Forms (WinForms)
*   **Библиотеки:**
    *   [Guna.UI2 WinForms](https://gunaui.com/) (v2.0.4.7): Использована для создания современного и стильного пользовательского интерфейса.
    *   [Newtonsoft.Json](https://www.newtonsoft.com/json) (v13.0.4): Применяется для сериализации данных в формат JSON и десериализации из него.

## Установка и запуск

### Предварительные требования
*   Visual Studio (любая версия, поддерживающая .NET Framework 4.7.2, например, Community 2019 или 2022).
*   Git (для клонирования репозитория).

### Шаги по установке

1.  **Клонируйте репозиторий:**
    Откройте терминал (командную строку) и перейдите в папку, куда хотите сохранить проект. Затем выполните команду:
    ```bash
    git clone https://github.com/Wa1ntGarner/ReportMaster.git
    ```

2.  **Перейдите в папку проекта:**
    ```bash
    cd ReportMaster
    ```

3.  **Откройте решение в Visual Studio:**
    Найдите и дважды кликните по файлу решения `ReportMaster2.sln` (или откройте Visual Studio и выберите **Файл -> Открыть -> Проект/Решение** и укажите путь к этому файлу).

4.  **Восстановите NuGet-пакеты:**
    При открытии проекта Visual Studio автоматически начнет восстанавливать необходимые NuGet-пакеты (`Guna.UI2` и `Newtonsoft.Json`). Если этого не произошло, нажмите правой кнопкой мыши на названии решения в **Обозревателе решений** и выберите пункт **"Восстановить пакеты NuGet"**.

5.  **Запустите приложение:**
    Нажмите клавишу `F5` или кнопку **"Пуск"** на панели инструментов Visual Studio. Приложение скомпилируется и запустится.
